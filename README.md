## Run Locally

Clone the project

```bash
  git clone https://github.com/Maure-dev/gerardiEcommerce.git
```

Go to the project directory

```bash
  cd gerardiEcommerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
